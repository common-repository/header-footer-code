== Description ==
This plugin help you easy to insert CSS, JavaScript and any codes to <head> or before </body>, for example <strong>Google Analytics</strong>, <strong>Facebook Pixel</strong>, <strong>custom CSS</strong>,  and much more. No need to edit your theme files, don't lost your code when upgrade your theme. This is must have plugin for your WordPress website.

If you to know how to add code to header in WordPress, or looking for <strong>wordpress plugin to add code to header</strong>, you go right place, this plugin help you do it.

<h4>INSTALLATION</h4>

Manual installation is easy and takes fewer than one minute.

1. Download the plugin from wordpress.org, unpack it and upload the <strong>[NinjaTeam Insert Code Header Footer]</strong> folder to your <strong>wp-content/plugins/</strong> directory.
2. Activate the plugin through the ‘Plugins‘ menu in WordPress.
3. Go to your main <strong>WordPress menu > Header Footer Code</strong> to add your custom code.

You’re done. Enjoy.


== Installation ==

1. Download the plugin from wordpress.org, unpack it and upload the <strong>[NinjaTeam Insert Code Header Footer]</strong> folder to your <strong>wp-content/plugins/</strong> directory.
